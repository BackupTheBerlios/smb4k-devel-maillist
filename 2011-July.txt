This is the first beta release of the upcoming Smb4K version 1.0.0. Changes 
compared to 0.10.69 comprise an improved handling of custom options (complete 
revision), improved notifications, a rewritten network neighborhood scanner 
(it uses KJobs now), and several small modifications and bug fixes.

For a detailed list of changes please refer to the ChangeLog file within the 
tarball.

This second beta release of Smb4K 1.0.0 comes with improvements to the browse 
code and the handling of 'homes' shares. The mounting under Linux has been 
fixed and initial work has been done to make Smb4K work properly under FreeBSD 
again.

Many thanks go to those users who helped us to improve Smb4K by sending bug 
reports. Please keep on reporting any bugs you find to our bug tracker: 
http://developer.berlios.de/bugs/?group_id=769
