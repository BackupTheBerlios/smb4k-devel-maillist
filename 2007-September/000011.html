Yesterday the Smb4K team released version 0.8.5 of the advanced network 
neighborhood browser for KDE. This release features an updated handbook and 
fixed compilation errors under FreeBSD.

ChangeLog:
- Updated the handbook.
- Fixed compile errors in Smb4KMounter class under FreeBSD.

The source code can be downloaded from 
http://prdownload.berlios.de/smb4k/smb4k-0.8.5.tar.bz2.
