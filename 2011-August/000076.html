This beta release comes with changes to the way mounpoints are stored 
internally (QString instead of QByteArray) and adds support for new arguments 
of the Samba utility programs. A regression compared to the 0.10.x series has 
been fixed where the smb.conf file was not read on start-up leading to default 
values not being set. Two crashes have been fixed in the preview dialog.
