With this development release you get the first alpha version of Smb4K 1.0.0. 
Compared to the 0.10 branch the program changed a lot. If you are interested 
in details, please refer to the ChangeLog file in the source tarball. The most 
important changes are:

- Improved integration with KDE.
- Usage of notifications where appropriate.
- Usage of KAuth to mount and unmount shares.
- Required minimum KDE version is 4.4.0.
- Initial laptop support through Solid.
- Most core classes use threads to execute their tasks.
- Improved handling of bookmarks.
- Editor for authentication information.
- Many usability enhancements.
- Translations from the KDE translators (l10n-kde4).

Since this is an alpha release, it most likely contains bugs. Please report 
them to our bug tracker.
